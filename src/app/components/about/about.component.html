Sobre Comic App

Estudihambre

Proyecto diseñado y contruido de la mano de Estudihambre , una compañia orgullosamente perteeciente a la ARCA (Alianza Rebelde del Codigo Abierto) por el fundador de dicha empresa Bryan Fernando Rangel Aguirre. Quien busca difundir el conocimientoe ideas innovadoras libremente a travez de aplicaciones, paginas web, videos ilustrativos, para de esta manera entre la comunidad desarrollar un proyecto totalmente funcional para los usuarios.

Angular JS

Framework desarrollado por Google.Proyecto de código abierto, realizado en Javascript que contiene un conjunto de librerías útiles para el desarrollo de aplicaciones web y propone una serie de patrones de diseño para llevarlas a cabo. En pocas palabras, es lo que se conoce como un framework para el desarrollo, en esta caso sobre el lenguaje Javascript con programación del lado del cliente.
